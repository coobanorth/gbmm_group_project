How To Use This Website

This website is an easy to use, fun way of teaching secondry school students about cyber security!

Each page includes easy to digest information about the given subject and a mini quit to test yourself at the end. Once you feel like you have learnt all there is to know about keeping safe and secure when using computers head over to our quiz page to see how uch you really know!

Use the glossary to learn the meanings of words you are not sure about. You can also access the glossary by using the information dropdown.

Who Is Behind This Website?

This website has been created by a group of university students passionate about sharing and getting the word out about staying safe online. From social media to storing files everything we do is vital money making information to somebody so we need to stay secure and safe!
